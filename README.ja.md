# 北陸観光移動軌跡

本プロジェクトは、北陸インバウンド観光DX・データコンソーシアムによって公開されているオープンデータ、金沢大学のWiFiパケットセンシングで取得したデータを可視化したものです。観光地の訪問状況や来訪者の移動軌跡を確認できます。

## デモ
- [北陸観光移動軌跡](https://code4fukui.github.io/hokuriku-kanko-route/)
- [北陸観光WiFiパケットセンシングによる時間帯別混雑状況目安](https://code4fukui.github.io/hokuriku-kanko-route/traffic.html)

## データ・API
- [hokuriku-inbound-kanko/mac-address](https://github.com/hokuriku-inbound-kanko/mac-address) - 金沢大学のWiFiパケットセンシングデータ
- [北陸インバウンド観光DX・データコンソーシアム | 観光DX](https://kanko-dx.jp/case-study/1784/) - オープンデータの情報源
- [TIFDATA 北陸観光データポータル](https://tifdata.jp/)
