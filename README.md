# hokuriku-kanko-route

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

This project visualizes open data published by the Hokuriku Inbound Tourism DX Data Consortium, specifically data obtained from WiFi packet sensing by Kanazawa University, using [data conversions](https://github.com/code4fukui/mac-address).

## Demo
- [Hokuriku Tourism Movement Trajectory](https://code4fukui.github.io/hokuriku-kanko-route/)
- [Hokuriku Tourism WiFi Packet Sensing Congestion by Time of Day](https://code4fukui.github.io/hokuriku-kanko-route/traffic.html)

## Features
- Visualization of Hokuriku tourism movement data
- Visualization of congestion level by time of day using WiFi packet sensing data

## Data / API
- [hokuriku-inbound-kanko/mac-address](https://github.com/hokuriku-inbound-kanko/mac-address) - WiFi packet sensing data from Kanazawa University
- [Hokuriku Inbound Tourism DX Data Consortium | Tourism DX](https://kanko-dx.jp/case-study/1784/) - Source of open data
- [TIFDATA Hokuriku Tourism Data Portal](https://tifdata.jp/)
